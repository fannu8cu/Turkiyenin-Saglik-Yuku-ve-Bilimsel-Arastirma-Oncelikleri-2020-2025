# Turkiyenin-Saglik-Yuku-ve-Bilimsel-Arastirma-Oncelikleri-2020-2025
Türkiye'nin 2020-2025 DALY (Hastalık Yükü) sıralaması ile bilimsel yayın öncelikleri (PubMed) arasındaki uyumu analiz eden bir veri bilimi projesi. 250k+ makale özetini sınıflandırmak için (TF-IDF + Logistic Regression) NLP modeli kullanılmıştır. Amaç: Veriye dayalı sağlık politikası önerileri sunmak.
